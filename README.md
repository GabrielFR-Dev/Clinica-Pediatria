# 📌 Sobre o Projeto

Aplicação Full Stack desenvolvida com foco em praticar conceitos modernos de desenvolvimento web, trabalhando desde a modelagem do banco de dados até o consumo de API no front-end.

O projeto simula uma landing page de uma clínica pediátrica, consumindo dados reais através de uma API REST desenvolvida em Node.js.

# ⚙️ Como Executar o Projeto

## Clone o repositório

```bash
git clone https://github.com/GabrielFR-Dev/Clinica-Pediatria
```

---

## Acesse a pasta

```bash
cd Clinica-Pediatria
```

---

## Instale as dependências

```bash
npm install
```

---

## Execute o projeto

```bash
npm run dev
```
